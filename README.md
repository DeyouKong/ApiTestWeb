## Environment

1. 首先安装  node  版本10-12即可; 确保 node -V
2. 安装 yarn 工具包
3. 命令都是在项目根目录下执行
```
npm install -g yarn


```

## 安装依赖
```
yarn install
```

## Develop 
    yarn serve

## Build
    yarn build
